# OneLedger 质量检查清单

复制到需求或 PR 中逐项勾选；不适用项写明原因，不能直接删除。

## 产品与内容

- [ ] 用户问题、范围和验收条件明确。
- [ ] 默认/空/加载/错误/禁用/成功状态完整。
- [ ] 用户文案具体、可行动，术语和金额格式一致。
- [ ] 没有无意扩大到云同步、AI 或其他非本期范围。

## UI 与原创性

- [ ] 使用 OneLedger token 和复用组件，无页面级硬编码品牌色。
- [ ] 紧凑布局仍保留至少 48dp 触控区。
- [ ] 深色、浅色、窄屏和字体放大无关键截断。
- [ ] 与参考产品在构图、颜色、图标、文案、动效中至少三项明显不同。
- [ ] 提供 Before / After / Why 和必要截图。

## 交互与无障碍

- [ ] 点击立即反馈，返回/取消不会丢失未提示的数据。
- [ ] 动画可中断、方向一致，并支持减少动态效果。
- [ ] 图标有语义描述，颜色不是唯一信息来源。
- [ ] TalkBack 顺序合理；手势功能有按钮替代。
- [ ] 删除、覆盖和清空操作明确影响并可确认/撤销。

## 数据与代码

- [ ] 金额使用 `Long` 最小货币单位，不使用 `Double`。
- [ ] UI 经 ViewModel/Repository 访问数据，不直接调用 DAO。
- [ ] 数据库变更包含 Migration 和迁移测试。
- [ ] 时间区间、跨日/月/年、空集合和极值已覆盖。
- [ ] 无密钥、真实财务数据、构建产物或无说明 TODO。

## 验证与发布

- [ ] `assembleDebug` 通过。
- [ ] `testDebugUnitTest` 通过。
- [ ] `lintDebug` 通过。
- [ ] 相关截图回归通过，基准变化已人工确认。
- [ ] PR 描述、规范/README、发布说明已同步。
- [ ] 已知限制和安全回滚方案已说明。

## 缺陷优先级

| 等级 | 定义 | 处理 |
| --- | --- | --- |
| P0 | 数据丢失、金额错误、隐私泄露、无法启动 | 停止发布，立即修复 |
| P1 | 核心记账/查询阻断，无法可靠绕过 | 当前版本修复 |
| P2 | 局部功能错误、明显无障碍或性能问题 | 进入最近迭代 |
| P3 | 轻微视觉、文案和非阻断体验问题 | 排期处理 |
