# 流畅导航与局部切换

## 目标

OneLedger 的页面切换需要保持空间连续、响应即时，并允许用户在动画未结束时继续操作。动效用于解释“内容从哪里来、回到哪里去”，不承担装饰任务。

## 交互规则

### 主 Tab

- 页面内容使用短交叉淡化，不做整屏横向飞入。
- 底栏只保留一个选中容器；容器沿底栏连续移动，快速连续点击时从当前位置改向。
- 离开 Tab 后保留可保存的页面状态，返回时不因重新创建而闪动或重置筛选。
- 图标和文字不切换字重，避免选中瞬间发生排版跳动。

### 账本子页面

- 从账本进入预算或收支日历：新页面从右侧低位移进入，账本向左低位移退出。
- 返回账本：完全镜像上述路径。
- 位移距离不超过页面宽度的 `1/12`，同时使用短淡化，避免“整屏飞页”。
- 底栏和快速记账按钮与页面层级同步淡出、下沉；返回首页时沿相反路径恢复。

### 页面内部

- 快速记账的支出、收入、转账内容使用交叉淡化，金额区和键盘保持稳定。
- 备注输入栏始终显示在金额下方，不再设置额外的“填写备注”入口。
- 点击备注栏立即聚焦并打开系统输入法；应用内数字面板同步淡出并收起，输入法完成后按相反路径恢复。
- 点击金额栏立即清除备注焦点、收起系统输入法并恢复应用内数字面板；金额与备注两种输入模式可随时来回切换，已输入内容均保留。
- 金额与备注切换由单一输入模式驱动：数字面板只做短淡化，禁止同时执行面板位移、布局伸缩和底部 Sheet 拖拽，避免与系统输入法动画叠加。
- 快速记账 Sheet 禁止整页纵向拖拽；分类网格保留一比一手势滚动，连续上下滑动不得带动 Sheet 抖动。
- 可滚动页面中的按压组件仅在实际按下和回弹期间创建缩放图层，静止滚动时不保留额外图形层。
- 统计范围使用单一移动指示器；范围数据更新不得触发整页重新入场。
- 日历选日只改变单元格底色，并对下方当日明细做短交叉淡化；日期网格尺寸不得变化。
- 收支日历的年月标题固定在月份内容之外，任何月份、有无账单或列表滚动位置都不得改变其顶部对齐。
- 当前月和相邻月的农历标签在后台预热；滑动期间按日期索引账单，不在每一帧遍历全部记录。

## 动效参数

| 用途 | 参数 |
| --- | --- |
| 选中颜色 | 140ms |
| 内容入场 | 220ms，标准减速 |
| 内容退场 | 140ms |
| 导航位移与移动指示器 | 临界阻尼、Medium stiffness |
| 按压 | 按下即时 `0.97`，释放使用临界阻尼 High stiffness |

## 性能与无障碍

- 动画只作用于必要的内容层，避免让整个 `LazyColumn` 重建后再次入场。
- 页面切换使用稳定内容键；底栏移动只动画一个指示器。
- 使用 Compose 的动画时钟，跟随 Android 系统“移除动画/动画时长缩放”设置。
- 动画不阻塞点击、返回、保存和连续切换。

## 验收标准

- 连续快速点击不同 Tab 时，底栏指示器从当前中间位置改向，没有瞬移或叠出多个高亮。
- 主 Tab 切换没有空白帧，返回后已保存的范围选择仍然存在。
- 账本子页面进入与返回方向相反，底栏不会突然消失或出现。
- 快速切换记账类型时，分类区无白闪，金额与键盘位置不跳。
- 日历连续点选日期时，网格完全不位移，下方内容可连续改向淡化。
- 连续滑动月份时，年月标题保持同一纵向基线，日历卡片顶部保持同一位置。
- 点击备注栏后，数字面板在 110ms 内开始退出，系统输入法可直接输入；页面中不存在重复的备注开关。
- 系统输入法显示时点击金额栏，输入法立即开始收起，数字面板沿原路径恢复且金额、备注内容不被重置。
- 账本“收支日历”的按压反馈必须在触点按下时出现，体感强度与“本月可用”预算卡一致；日历农历数据不得阻塞首次页面反馈。
- 目标设备常规操作维持 60fps；静态截图与动画前后的最终状态一致。
